Инструменты и технологии

которые я использую
Html5
HTML — Cтандартизированный язык гипертекстовой разметки документов. Используется для создания каркаса сайта, то-есть вся разметка и весь текст, размещаются с его помощью. При написании кода в HTML, использую технологию "БЭМ" c ее помощью можно использовать повторно различные компоненты, расширять их, избегая дубликата кода.

Так же в работе при необходимости могу использовать библиотеку Bootstrap.

ПОДРОБНЕЕ... НАЗАД...
CSS3
CSS — Формальный язык описания внешнего вида документа или же язык стилей. С помощью CSS простой набор текста и картинок, привращяется в полноценный сайт. Отлично знаю все технологии CSS. На чистом CSS не пишу, использую препроцессоры SASS или SCSS (его больше), вместе с плагином для GULP. Так же придерживаюсь методологии БЭМ.


ПОДРОБНЕЕ... НАЗАД...
JavaScript
JavaScript — Мультипарадигменный язык программирования. С помощью JS сайт становится "живым" и отзывчивым к пользователю. Знаю на базовом уровне, в процессе изучения...

ПОДРОБНЕЕ... НАЗАД...
REACT
React — JavaScript-библиотека для разработки пользовательских интерфейсов. В основном используется для создания Web-приложения, и крупных проектов. Знаю на начальном уровне, в процессе изучения...

ПОДРОБНЕЕ... НАЗАД...
Vue.Js
Vue.js — JavaScript-фреймворк для создания пользовательских интерфейсов. Так же в основном используется для создания Web-приложения, и крупных проектов. Знаю на начальном уровне, в процессе изучения...

ПОДРОБНЕЕ... НАЗАД...
jQuery
jQuery — Набор функций JavaScript. С ее помощью можно легко взаимодействовать с HTML элементами. Знаю на базовом уровне, в процессе изучения...
